## 透明锁屏

## 权限说明  
* **程序在锁屏时，需要通过注册表屏蔽掉任务管理器，因此需要管理员权限运行。**  
* **程序通过 user32 设置全局快捷键、禁用部分按键，因此部分杀毒软件会报病毒。**
